### PROJECT NAME: Intelligent Neuro-Evolutionary Hybrid AI Framework for Anomaly Detection, Failure Risk Assessment, and Predictive Maintenance in Industrial Systems
• Description: In this project, we developed an intelligent predictive maintenance system using the AI4I 2020 dataset, integrating ML and Deep Learning and Genetic Algorithms. The model performs anomaly detection,
failure prediction, and risk assessment, enabling early fault identification and optimized maintenance decisions, improving system reliability and reducing downtime.
• Tools and Technologies: Python, Pandas, NumPy, Scikit-learn, TensorFlow (Keras), XGBoost,
DEAP (Genetic Algorithm), SMOTE, Matplotlib, Seaborn, SHAP, Joblib
