### PROJECT NAME: Predicting Machine Failures Using Genetic Algorithms, Machine Learning & Deep Learning Algorithms (Neural Networks) for Sustainable Manufacturing and Service Economics
#-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
• Description:->
(i) This project study uses/Worked with a Predictive Maintenance Dataset from the UCI repository to predict and classify machine failures.The study has one objectives: checking deep learning methods for better accuracy.This study explores the integration of Genetic Algorithms (GA) with Deep Learning (DL) algorithms for predictive maintenance using the AI4I 2020 Predictive Maintenance Dataset.
(ii) In this project, we developed an intelligent predictive maintenance system using the AI4I 2020 dataset, integrating ML and DL and GA. The model performs anomaly detection, failure prediction, and risk assessment, enabling early fault identification and optimized maintenance decisions, improving system reliability and reducing downtime.
• Tools Used: MATLAB/Python, Industry 4.0, DEAP(Genetic Algorithms (G.A.), SMOTE), Machine Learning (ML), Deep Learning (Neural Networks), Hyperparameter tuning, Predictive maintenance, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, TensorFlow (Keras), XGBoost Classifier, SHAP, Joblib, Jupyter Notebook
