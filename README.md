### PROJECT NAME: Predicting Machine Failures Using Genetic Algorithms, Machine Learning & Deep Learning (Neural Networks) Algorithms for Sustainable Manufacturing and Service Economics
#-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
• Description:->>
  Worked with Intelligent Neuro-Evolutionary Hybrid AI Framework for Anomaly Detection, Failure Risk Assessment, and Predictive Maintenance in Industrial Systems.In this project, we developed an intelligent predictive maintenance system using the AI4I 2020 dataset, integrating ML and Deep Learning and Genetic Algorithms. The model performs anomaly detection, failure prediction, and risk assessment, enabling early fault identification and optimized maintenance decisions, improving system reliability and reducing downtime.
